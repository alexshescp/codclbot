# Codcl.com After-AI Telegram Bot

Телеграм-бот для расчета стоимости доработки AI/No-Code прототипов.
Проект демонстрирует архитектуру сервисов на Go, обработку многошаговых сценариев, отправку изображений, сбор контактов и передачу заявок администратору.

## Основные возможности

### Многошаговый калькулятор

Бот последовательно уточняет параметры проекта:

* состояние фронтенда,
* тип backend-архитектуры,
* количество функций,
* выбор базы данных,
* выбор инфраструктуры.

Каждый шаг снабжен пояснениями, отображаемыми перед кнопками.

### Поддержка изображений

Стартовое сообщение и информационные разделы отправляются в формате:

* изображение,
* подпись (Caption) с Markdown-разметкой,
* кнопки.

### Оставление контактов

Бот принимает контакт в любой форме (email, номер телефона, @username или произвольный текст), ограничивает количество отправок до 10 в сутки для каждого пользователя и отправляет заявку администратору.

### Админский функционал

Все контакты пересылаются в чат администратора, идентификатор которого задается в `.env`.

### Технические особенности

* Long polling
* Поддержка graceful shutdown
* Health check endpoint
* Структура каталогов, ориентированная на промышленное применение

## Требования

* Go версии 1.24 или выше.
* Созданный бот в Telegram и валидный токен.
* Администратор должен один раз написать боту, чтобы узнать свой chat_id.

## Настройка окружения

Создайте файл `.env` в корне проекта:

```
TELEGRAM_TOKEN=123456:ABCDEF
ADMIN_CHAT_ID=987654321
DEBUG=true
HEALTH_PORT=8080
```

### TELEGRAM_TOKEN

Токен создается через BotFather.

### ADMIN_CHAT_ID

Идентификатор чата администратора.
Варианты получения:

* временная команда `/whoami`,
* вызов API `getUpdates`,
* вывод chatID в лог при первом сообщении.

## Как адаптировать проект под свои задачи

### 1. Изменить текстовое содержание

Редактируйте файлы:

```
internal/services/messaging/*.go
```

### 2. Обновить прайсинг

Правьте JSON-файлы:

```
internal/config/frontend.json
internal/config/backend.json
internal/config/database.json
internal/config/infrastructure.json
```

### 3. Заменить изображения

Положите свои изображения в папку `assets` и обновите пути в handlers.

### 4. Настроить логику обработки контактов

Изменяйте:

```
internal/bot/handlers/contact.go
```

---

## Как запустить свой собственный бот на основе проекта

1. Создайте бота через BotFather.
2. Склонируйте репозиторий.
3. Установите Go 1.24+.
4. Выполните `go mod tidy`.
5. Настройте `.env`.
6. Запустите командой `go run ./cmd/bot`.
7. Перепишите текст, прайсинг и изображения под свои задачи.
